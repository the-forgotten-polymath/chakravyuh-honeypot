# API Key Documentation

## Generate API Key

Run the generator script:

```bash
python generate_api_key.py
```

Example output:

```
lo3jhse9mf6jqzug76t20x0g522xyxns6y3y

# .env file
API_KEY=lo3jhse9mf6jqzug76t20x0g522xyxns6y3y

# HTTP request header
X-API-Key: lo3jhse9mf6jqzug76t20x0g522xyxns6y3y
```
